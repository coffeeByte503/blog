Iniciando con canvas

Canvas se usa para dibujar gráficos que incluyen todo, desde líneas simples hasta objetos gráficos complejos

Canvas es una cuadrícula bidimensional, la esquina superior izquierda del lienzo tiene coordenadas (0,0).
Las coordenadas X aumentan a la derecha.
La coordenada Y aumenta hacia la parte inferior del lienzo.

Para empezar a usar canvas necesitamos agregar la etiqueta <canvas></canvas> a nuestro html

se puede crear y agregar al DOM desde javascript, pero eso lo veremos más adelante.

Nuestro HTML debe verse así:

<canvas id="canvas"></canvas>

Nota que he agregado el atributo id que nos servirá para seleccionar el elemento canvas con js y almacenarlo en una constante.

Nuestro js se verá asi:

var canvas = document.getElementById.("canvas");

Ahora ya podemos manipular los estilos y dimensiones del lienzo, pero no podemos dibujar nada aún.

Para ello debemos de obtener el contexto del lienzo para poder dibujar en él, el contexto nos proveerá de métodos y propiedades para hacer lo que nuestra imaginación nos permita.

Si te cuesta entender imagina lo siguiente: el lienzo es un trozo de papel y el contexto es un lápiz, es decir que utilizamos el lápiz para dibujar en el papel (es más que obvio pero aveces las personas no entienden) La constante canvas solo va hacer usada para redimensionar el papel, mejor dicho adaptarlo a la pantalla o ajustarlo a nuestro gusto.

El método getContext("2d") nos devolverá un objeto el cual guardaremos en una, constante para poder manipular el canvas.

var context = canvas.getContext("2d");

El objeto context

El objeto context nos provee varios métodos para dibujar.

  • fillRect
  • rect
  • arc
  • cuadratic
  • stroke
  • fill
  • moveTo
  • lineTo
  • quadricCurveTo
  • bezierCurveTo
  • arcTo
  • drawImage
  • fillText
Algunas propiedades:
  • textAlign
  • globalAlpha
  • fillStyle
  • strokeStyle
  • lineWidth

Estas son algunas de las propiedades y metodos, hay muchas mas. A lo largo del curso las iremos usando

Primeros dibujos

Usando el método fillRect:

El método fillRect toma los siguientes parámetros (x, y, w, h)

x & y son las coordenadas de nuestro canvas , w & h es el ancho y largo de la figura que vamos acrear

Usando el método rect:

Al igual que .fillRect(), rect toma los siguientes parámetros (x, y, w, h)


A diferencia de .fillRect(), .rect() solo dibuja los bordes de la figura sin rellenarlo. Nota tambien que he usado:
.beginPath()
.closePath()
.stroke()

1- El método beginPath() comienza una nueva ruta vaciando la lista de subrutas. Llame a este método cuando desee crear una nueva ruta.

2- Usamos rect() para crear un camino (un cuadro en este caso)

3- Usamos closePaht()
Este método no dibuja nada directamente en el lienzo. El método crea una ruta desde el punto actual hasta el punto de partida, es decir .beginPath()

4- Finalmente usamos el método stroke() para dibujar realmente (renderizar) la figura en el lienzo.

Cambiando los estilos

El objeto context también nos provee de propiedades para manipular colores, fuentes, opacidad etc.

Cambiendo el color del trazo con strokeStyle:
context.strokeStyle = "#000";
Cambiando el relleno con fillStyle:
context.fillStyle = "red";

podemos asignar un color en exadecimal, rgb, rgba, nombre de color, etc.
