Getting started with Canvas

The html canvas is used to draw graphics that include everything from simple lines to complex graphics objects.

The html canvas is two-dimensional grid, the upper-left corner of the canvas has coordinates(0,0).
X coordinates increases to the right.
Y coordinate increases toward the bottom of the canvas.

To start using canvas we need to add the <canvas></canvas> tag to our html

It can be created and added to the DOM from javascript, but we will see later.

Our HTML should look like this:

<canvas id="canvas"></canvas>

Notice that I have added the id attribute that will be used to select the canvas element with js and store it in a constant.

Our js will look like this:

var canvas = document.getElementById.("canvas");

Now we can manipulate the styles and dimensions of the canvas, but we can't draw anything yet.

For this we must obtain the context of the canvas to be able to draw on it, the context will provide us of methods and properties to do what our imagination allows us.

If you have trouble understanding imagine the following: the canvas is a piece of paper and the context is a pencil, that is, we use the pencil to draw on the paper (it is more than obvious but sometimes people do not understand) The variable canvas only goes make used to resize the paper, rather adapt it to the screen or adjust it to our way.

El método getContext("2d") nos devolverá un objeto el cual guardaremos en una, constante para poder manipular el canvas.

var context = canvas.getContext("2d");

El objeto context

El objeto context nos provee varios métodos para dibujar.

  • fillRect
  • rect
  • arc
  • cuadratic
  • stroke
  • fill
  • moveTo
  • lineTo
  • quadricCurveTo
  • bezierCurveTo
  • arcTo
  • drawImage
  • fillText
Algunas propiedades:
  • textAlign
  • globalAlpha
  • fillStyle
  • strokeStyle
  • lineWidth

Estas son algunas de las propiedades y metodos, hay muchas mas. A lo largo del curso las iremos usando

A diferencia de .fillRect(), .rect() solo dibuja los bordes de la figura sin rellenarlo. Nota tambien que he usado:

1- El método beginPath() comienza una nueva ruta vaciando la lista de subrutas. Llame a este método cuando desee crear una nueva ruta.

2- Usamos rect() para crear un camino (un cuadro en este caso)

3- Usamos closePaht()
Este método no dibuja nada directamente en el lienzo. El método crea una ruta desde el punto actual hasta el punto de partida, es decir .beginPath()

4- Finalmente usamos el método stroke() para dibujar realmente (renderizar) la figura en el lienzo.

Cambiando los Colores

El objeto context también nos provee de propiedades para manipular colores, fuentes, opacidad etc.

Cambiendo el color del trazo con strokeStyle:
context.strokeStyle = "#000";
Cambiando el relleno con fillStyle:
context.fillStyle = "red";

podemos asignar un color en exadecimal, rgb, rgba, nombre de color, etc.
